# 改动日志

## 2026-06-10：Docker CI + GitHub Pages 部署

### 一、GitHub Actions CI（Docker 冒烟测试）

**新增文件：** `.github/workflows/ci.yml`

每次 push 到 `main` 分支或提交 PR 时，自动在 GitHub Runner 上执行：

1. 从 Zenodo 下载 KuaiRec 真实数据（~1.5GB）
2. 启动 Docker Compose 集群（10 个容器：HDFS + YARN + Spark + Hive + MySQL）
3. 验证 HDFS 连通性（`dfsadmin -report`）
4. 上传数据到 HDFS
5. 验证 Hive Metastore（`SHOW DATABASES`）
6. 验证 Spark Master UI（`:8080`）
7. 运行基础 Hive 查询测试
8. 自动清理容器

**涉及修改：**
- `.gitignore` —— 新增忽略 `结果数据/`、`临时/`、`*.zip`

### 二、GitHub Pages 可视化预览

**新增文件：** `docs/index.html`

将 `可视化源码/preview.html` 复制到 `docs/index.html`。

这是一个纯静态页面，内置 ECharts 6 张图表 + 模拟数据，无需任何后端服务器即可运行。

**启用方式（需手动操作）：**
1. 打开 GitHub 仓库 → Settings → Pages
2. Source 选择「Deploy from a branch」
3. Branch: `main`, Folder: `/docs`
4. 点击 Save
5. 等待 1-2 分钟后访问 `https://<用户名>.github.io/<仓库名>/`
